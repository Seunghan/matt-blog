### 네트워크란 
네트워크  기본적으로 LAN과 WAN 으로 구분된다. LAN은 직접 관리할 수 있는 가까운 범위를 의미하고 WAN은 현관 밖에 있는 도로를 WAN이라고 할 수 있다.

### 네트워크의 표준화 개념
네트워크를 구축하고 설계할 수 있는 표준화된 개념으로 OSI 참조 모델이 있다.
**OSI**란 **(Open Systems Interconnection)**이라는 뜻으로 국제표준화기구에서 제정한 네트워크 통신 표준 모델이다.

OSI는 다수의 프로토콜을 역할별로 분류하여 명확하게 하는 모델이다. 
기본적으로 7 Layer를 가지고 있으며 **데이터 전송 기능**은 1~3계층, 이를 **보완하는 기능**은 4계층에서, **통신 서비스 기능**은 5~7계층에서 담당한다.

위 그래프는 사실 컴퓨터 공학에서나 배울만한 그림이기는 하다. 비개발자의 시선에서 조금 풀어서 설명하면 다음 테이블을 참고해 주면 된다.

OSI 참조 모델을 택배에 비유를해보면 된다.

| 계층명                | 계층 역할                                     |
| ------------------ | ----------------------------------------- |
| 응용 계층              | 과자를 선택한다                                  |
| 표현 계층              | 택배 상자에 들어가도록 과자량을 조절한다                    |
| 세션 계층              | 택배 송장에 받는 사람과 보내는 사람을 기입한 후               |
| 전송 계층              | 도착 예정일에 친구가 없으면<br>(친구에게 미리 택배가 간다고 알려준다) |
| 네트워크 계층            | 택배 업자는 배송 트럭이나 경로를 결정하고                   |
| 데이터 링크 계층<br>물리 계층 | 제대로 상대방에게 도달하도록 트럭에 화물을 싣는다               |

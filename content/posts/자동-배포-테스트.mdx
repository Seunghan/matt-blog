# 자동 배포 테스트

Obsidian에서 글을 작성하면 GitHub Action을 통해 자동으로 블로그에 배포됩니다.

## 흐름

1. Obsidian에서 `Blog/` 폴더에 글 작성
2. Obsidian Git 플러그인이 자동 push
3. GitHub Action이 `matt-blog` 레포에 동기화
4. Vercel이 자동 배포

잘 보이면 성공입니다!
